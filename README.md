# Index.html

Projeto simples para publicar uma página estática usando GitHub Pages.

Descrição
- Página de exemplo com "Olá, GitHub Pages!" em Português.
- O conteúdo principal da página está em `index.html`.

Como ver a página
1. Habilite GitHub Pages nas configurações do repositório (se ainda não estiver habilitado).
2. Coloque `index.html` na raiz do repositório.
3. Acesse a URL do GitHub Pages do repositório (algo como `https://<seu-usuario>.github.io/Index.html/`).
